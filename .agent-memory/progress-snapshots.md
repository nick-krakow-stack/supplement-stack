# Progress Snapshots

This file is reserved for automatic Stop and PreCompact snapshots when recent
turn/task progress must survive context compression.

Keep snapshots short and focused on completed work, open work, next steps, and
checks/status. Move durable completed project state into
`.agent-memory/current-state.md`, exact continuation context into
`.agent-memory/handoff.md`, and durable priorities into
`.agent-memory/next-steps.md`.
