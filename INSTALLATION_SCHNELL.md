# 🚀 Supplement Stack PHP - Schnelle Installation

## 📦 Für All-Inkl Privat Plus Server

### ⚡ Schnell-Installation (3 Schritte):

1. **Alle Dateien per FTP** in dein Webroot-Verzeichnis hochladen
2. **Browser öffnen:** `https://deine-domain.de/install_simple.php`
3. **Formular ausfüllen** und Installation starten

### 🔧 Was du benötigst:
- All-Inkl MySQL-Datenbank (bereits angelegt)
- FTP-Zugang zu deinem Webserver
- 5 Minuten Zeit

### 🏥 Bei Problemen:
1. **Debug-Script:** `https://deine-domain.de/debug.php`
2. **System-Check:** Zeigt alle PHP-Extensions und Berechtigungen
3. **Fehleranalyse:** Detaillierte Informationen über mögliche Probleme

### 📋 MySQL-Daten bereithalten:
```
Host: mysql5.all-inkl.com (oder ähnlich)
Datenbank: dein_db_name
Benutzer: dein_username  
Passwort: dein_passwort
```

### ✅ Nach Installation:
- **Installer löschen:** `install.php` und `install_simple.php` entfernen
- **Login:** `https://deine-domain.de/login.php`
- **Dashboard:** Erste Supplements hinzufügen
